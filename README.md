# Void Defender

## Project Overview
Void Defender is an exhilarating space-themed action game where players must protect their home planet from invading alien forces. With stunning graphics and intuitive gameplay, players can utilize various strategies and weapons to fend off relentless waves of enemies.

## Features
- **Dynamic Gameplay**: Experience fast-paced action with different game modes and difficulty levels.
- **Stunning Graphics**: Immersive 3D environments and beautifully animated characters and effects.
- **Customizable Spacecraft**: Upgrade your ship with various skins, weapons, and defensive components.
- **Co-op Multiplayer**: Team up with friends and tackle challenges together in cooperative mode.
- **Leaderboards**: Compete with players around the world for the highest scores and achievements.
- **Progressive Difficulty**: Face increasingly challenging waves of enemies as you advance.
- **Power-ups and Bonuses**: Collect power-ups to boost your abilities temporarily.

## Game Modes
- **Story Mode**: Follow an engaging narrative through multiple levels.
- **Survival Mode**: Survive as many waves as you can.
- **Challenge Mode**: Complete specific objectives under time pressure.
- **Multiplayer Arena**: Battle against other players in real-time combat.

## Installation

### Prerequisites
- Node.js (v14.0.0 or higher)
- npm or yarn package manager

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/shariquekhan1997/Void-Defender.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Void-Defender
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

### Development
To start the development server:
```bash
npm start
```
The game will open in your default browser at `http://localhost:3000`.

### Build for Production
To create an optimized production build:
```bash
npm run build
```

### Run Tests
To run the test suite:
```bash
npm test
```

## Controls
- **Arrow Keys**: Move your spacecraft
- **Space Bar**: Fire primary weapon
- **Shift**: Activate shield/dodge
- **P**: Pause/Resume
- **ESC**: Return to menu

## Configuration
Edit the `.env.local` file to customize game settings:
```env
REACT_APP_GAME_SPEED=1.0
REACT_APP_DIFFICULTY=medium
REACT_APP_ENABLE_SOUND=true
```
